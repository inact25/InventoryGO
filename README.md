# InventoryGO
Tech Inventory App using Golang and Console

## create config file

create [config.env], and put it into root of your project folder.Then fill it with :
```env
DbEngine = 'mysql'
DbUser = 'root'
DbPass = 'yourPass'
DbHost = 'localhost'
DbPort = 'yourPort'
DbConnection = '@tcp'
DbSchema = 'yourDbName'
```
